# 图标资源说明

按《协作规范》与《设计规范》，图标统一使用 `images/icon/` 目录下的 PNG 文件。

## TabBar 底部导航栏

| 文件名 | 用途 | 建议尺寸 | 颜色 |
|--------|------|----------|------|
| home.png | 首页-未选中 | 81×81 | #A0A0A0 |
| home_active.png | 首页-选中 | 81×81 | #EAB308 |
| camera_tab.png/camera_tab_active.png | 定损-未选中/选中 | 81×81 | #A0A0A0 / #EAB308 |
| person.png | 我的-未选中 | 81×81 | #A0A0A0 |
| person_active.png | 我的-选中 | 81×81 | #EAB308 |

## 首页已使用

| 文件名 | 用途 | 建议尺寸 | 颜色 |
|--------|------|----------|------|
| camera.png | AI 定损入口 | 48×48 @2x | **白色**（适配蓝渐变背景） |
| search.png | 搜索框 | 28×28 @2x | #94A3B8 或 #9CA3AF |
| location.png | 维修厂地址 | 24×24 @2x | #94A3B8 或 #9CA3AF |

## 若图标不存在

代码中已写明路径，若文件缺失请按上表下载或制作。TabBar 图标建议 81×81px（微信规范）。
