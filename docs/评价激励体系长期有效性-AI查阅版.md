## 二、底层原则（不可突破）

| 原则 | 说明 |
|------|------|
| 中立性优先 | 好评差评规则一致；评价收益与门店解绑，锚定「内容对用户的决策价值」 |
| 价值与收益对等 | 内容价值、决策影响与奖金正相关，上不封顶 |
| 盈利绝对可控 | 奖金从佣金列支；常规≤80%；转化奖金单独列支，零垫资 |
| 全体系一致性 | 增量延伸，复用复杂度分级、有效评价、账号权重 |
| 防刷前置 | 权重分层、滞后结算、违规追溯 |
| **终身可追溯** | 全程留痕不可篡改，违规一经核实即可追回全部奖金、执行处罚 |

## 三、核心术语

| 术语 | 定义 |
|------|------|
| 有效评价 | 交易闭环、凭证满足、有相关描述、账号合规；可参与点赞追加核算 |
| 有效点赞 | 停留≥30 秒、账号权重＞0、单用户单评价终身 1 次、无刷赞异常 |
| 账号综合权重系数 | 基础可信度权重 × 车型匹配权重 |
| 7 天决策窗口期 | 点赞后 7 日内用户成交，可触发该评价的内容转化奖金 |
| 内容转化奖金池 | 每笔成交佣金 50% 划出，仅分配给影响决策的有效评价发布者 |
| **事后验证有效点赞** | 下单前已浏览、修完车后回头点赞，适用专属加权与补发 |
| **内容转化效率分** | （7 天内下单数 ÷ 总浏览数）× 100% × 同项目匹配系数；同车型同项目才计入 |

## 四、三大奖金模块

### 4.1 模块一：常规点赞追加

```
单条评价当月常规点赞追加奖金 = 订单实收佣金 × 0.5% × 当月新增有效点赞的「账号综合权重系数」之和
```

| 基础可信度权重 | 4 级 2.0 / 3 级 1.0 / 2 级 0.5 / 0-1 级 0 |
| 车型匹配权重 | 精准 2.0 / 高度 1.5 / 基础 1.0 / 无 0.5 |
| 收益有效期 | 2 年；3 级及以上标杆/爆款追评可延长 1 年 |

### 4.2 模块二：内容转化追加

```
单条评价可获得的单笔成交转化奖金 = 内容转化奖金池 ×（该评价决策权重 ÷ 本次成交所有有效评价决策权重总和）
```

**决策权重** = 决策时间 × 内容停留 × 内容匹配 × 内容价值（四维相乘）

**准入**：有效评价 + 7 天窗口内有效点赞 + 车型/项目匹配 + 单笔最多 10 条分配

### 4.3 模块三：事后验证有效点赞

**判定（5 项全满足）**：下单前 7 天有浏览 + 同品牌同车系同类项目真实交易 + 完工后 30 天内点赞 + 有效点赞标准 + 单笔交易最多 1 条触发

**前台**：排序权重×2；≥10 次打「车主验证・真实避坑」红标；单独展示「XX 位车主验证有用」；账号成长分加成

**后台**：常规点赞全额核算；转化补发：该笔成交未给其他评价分配过时，从奖金池划出 50% 补发

## 五、封顶与结算

| 规则 | 说明 |
|------|------|
| 可叠加 | 基础 + 追评 + 常规点赞追加 + 内容转化追加 + 事后验证专属补发 |
| 常规封顶 | 基础 + 追评 + 常规点赞追加 ≤ 实收佣金 80% |
| 超额无上限 | 内容转化追加、事后验证补发不纳入封顶 |
| 实时反馈 | 发布后实时显示基础奖励、预计追加、成长进度；每收到点赞实时更新奖金 |
| 结算 | 基础奖励审核通过即可解冻提前提现；**常规点赞追加月度核算已取消**，发放方式待定 |

## 六、四维一体反作弊（事前免疫）

| 维度 | 校验 |
|------|------|
| 设备 | 唯一性、黑产库、多账号；一设备一主账号 |
| 账号 | 实名、车辆、可信度、违规；一人一证一号 |
| 行为 | 浏览/点赞/评价/下单路径；秒赞、秒下单、批量操作拦截 |
| 交易 | 真实性、商户关联、支付异常；高频关联冻结核查 |
