## 一、权威文档一览（冲突时以此为准）

| 领域 | 权威文档（唯一依据） | 衍生/速查文档 | 同步要求 |
|------|----------------------|---------------|----------|
| **核心算法** | `全指标底层逻辑梳理.md` | 无 | 算法变更必更 |
| **用户等级 0-4 级** | `用户等级体系.md` | `docs/用户等级体系-AI查阅版.md`、`.cursor/rules/用户等级体系.mdc` | **三处必须同步** |
| **评价/合规/防刷** | `评价和激励体系.md` | 无 | 合规规则变更必更 |
| **点赞追加奖金** | `点赞追加奖金体系完整标准方案-V1.0正式版.md` | `docs/评价激励体系长期有效性-AI查阅版.md`、`docs/点赞追加奖金体系-开发计划.md` | **权威 + AI 查阅版必须同步**；开发计划按任务变更更新 |
| **奖励金后台配置** | `全指标底层逻辑梳理.md`（算法）+ `docs/评价奖励金体系-设计方案.md`（配置项） | 无 | 参数以全指标为准 |

## 二、冲突解决规则

1. **用户等级**：`用户等级体系.md` 为唯一权威；`docs/用户等级体系-AI查阅版.md`、`.cursor/rules/用户等级体系.mdc` 与权威不一致时，**以权威为准并立即同步**。
2. **点赞追加奖金**：`点赞追加奖金体系完整标准方案-V1.0正式版.md` 为唯一权威；`docs/评价激励体系长期有效性-AI查阅版.md` 为速查版，冲突时以权威为准。
3. **核心算法**：`全指标底层逻辑梳理.md` 为唯一权威，评价权重、店铺得分、奖励金公式、防刷铁律均以此为准。
