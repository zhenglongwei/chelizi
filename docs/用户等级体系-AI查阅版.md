> **权威来源**：`用户等级体系.md`。本体系为平台全链路规则的核心底层支撑，贯穿评价权重、点赞奖金、奖励金领取、内容排序、功能准入。**与评价体系、点赞追加奖金、竞价分发、防刷合规 100% 兼容，冲突时以本体系为准。**

## 一、快速速查（TL;DR）

| 等级 | 名称 | 评价权重 | 点赞可信度权重 | 核心门槛 |
|------|------|----------|----------------|----------|
| 0 | 风险受限 | 0 | 0 | 一票否决项触发 |
| 1 | 基础注册 | 0.3 | 0.3 | 实名+车辆绑定+无违规 |
| 2 | 普通可信 | 1.0 | 1.0 | 注册≥7天、≥2笔交易、≥2条有效评价 |
| 3 | 活跃可信 | 2.0 | 1.5 | 注册≥30天、≥5笔交易、≥3条优质评价、100%合规 |
| 4 | 核心标杆 | 3.0 | 2.0 | 注册≥90天、≥10笔交易、≥5条标杆评价、≥100同车型点赞 |

**核算方式**：**实时核算**（每次请求时根据实名、车辆、订单、评价、违规等动态计算）。**不可跳级**，必须逐级满足。*注：月度批量核算已取消。*

---

## 二、设计原则（不可突破）

- **真实行为唯一驱动**：等级仅由真实交易、真实评价、合规行为决定，无付费升级、无人工干预
- **全链路强绑定**：等级决定话语权、收益、权限
- **防刷前置兜底**：低等级无核心话语权、无高额收益，高等级必须长期真实行为沉淀
- **动态可追溯**：实时核算，升降级留痕，违规终身追溯
- **汽修场景专属**：无真实车辆绑定、无真实维修交易，永远无法高权重

## 三、等级准入与保级标准（系统自动核算）

### 3.1 0 级 - 风险受限账号

| 项 | 内容 |
|----|------|
| 准入 | 触发一票否决项直接进入 |
| 保级 | 无；完成实名+车辆绑定后可升至 1 级 |
| 永久锁定 | 有重大违规记录永久 0 级 |

### 3.2 1 级 - 基础注册账号

| 项 | 内容 |
|----|------|
| 准入 | ① 手机号实名+身份证真人核验 ② 车辆行驶证核验绑定 ③ 无违规 |
| 保级 | 实名+车辆有效，无重度及以上违规 |
| 永久锁定 | 未完成有效交易永久 1 级 |

### 3.3 2 级 - 普通可信账号

| 项 | 内容 |
|----|------|
| 准入 | ① 满足 1 级 ② 注册≥7 天 ③ 累计≥2 笔有效交易 ④ 累计≥2 条有效评价 ⑤ 合规率≥90% ⑥ 无中度及以上违规 |
| 保级 | 近 3 月≥1 笔有效交易；月度合规率≥80%；无重度及以上违规 |
| 永久锁定 | 累计 2 次中度违规，永久上限 2 级 |

### 3.4 3 级 - 活跃可信账号

| 项 | 内容 |
|----|------|
| 准入 | ① 满足 2 级 ② 注册≥30 天 ③ 累计≥5 笔有效交易 ④ 累计≥3 条 2 级及以上优质评价 ⑤ 合规率 100% ⑥ 终身无中度及以上违规 |
| 保级 | 近 3 月≥2 笔有效交易；月度合规率≥90%；无中度及以上违规 |
| 永久锁定 | 出现 1 次中度违规，永久取消 3 级及以上资格 |

### 3.5 4 级 - 核心标杆账号

| 项 | 内容 |
|----|------|
| 准入 | ① 满足 3 级 ② 注册≥90 天 ③ 累计≥10 笔有效交易 ④ 累计≥5 条 3 级及以上标杆评价 ⑤ 评价获≥100 同车型有效点赞 ⑥ 合规率 100% ⑦ 终身零违规 |
| 保级 | 近 3 月≥3 笔有效交易；月度合规率 100%；无任何轻度及以上违规 |
| 永久锁定 | 出现 1 次任何违规，永久取消 4 级资格，降至 2 级及以下 |

## 四、一票否决项（直接 0 级，无申诉）

- 未完成实名认证、车辆绑定，或实名/车辆信息伪造、冒用
- 有重大违规记录，或累计 2 次及以上重度违规
- 同设备/身份证/手机号关联多账号，除主账号外其余永久 0 级
- 系统识别为批量注册小号、机器号、水军号，无真实交易

---

## 五、违规扣减与等级影响

| 违规等级 | 扣减 | 等级影响 |
|----------|------|----------|
| 轻度（水评、无凭证、无效举报） | 单次扣 50% 成长分，限制 7 天 | 无法升级；累计 2 次→中度 |
| 中度（虚假评价、恶意差评、刷赞、秒赞） | 扣 100% 当月成长分，权重清零 30 天，追回奖金 | 直接降 1 级；累计 2 次→重度 |
| 重度（刷单、合单拆单、批量刷量、工作室作弊） | 扣全部历史成长分，追回所有奖金 | 直接降至 0 级，永久限制 |
| 重大（伪造凭证、辱骂诽谤、涉嫌违法） | 一票否决 | 永久封禁，0 级 |

## 六、全链路权益与权重（唯一依据）

### 6.1 评价权重系数

| 等级 | 系数 | 规则 |
|------|------|------|
| 0 | 0 | 仅个人可见，不计入店铺评分 |
| 1 | 0.3 | 仅计入基础分，不影响排序，无浮动奖励 |
| 2 | 1.0 | 全额计入，正常排序，可领全额奖励 |
| 3 | 2.0 | 权重翻倍，默认置顶，奖励上限+50% |
| 4 | 3.0 | 权重 3 倍，永久置顶，差评额外翻倍，奖励上限+100% |

### 6.2 点赞追加奖金可信度权重

| 等级 | 权重 | 规则 |
|------|------|------|
| 0 | 0 | 不产生奖金，不纳入核算 |
| 1 | 0.3 | 仅基础点赞奖金，无转化加成 |
| 2 | 1.0 | 全额纳入，可触发内容转化奖金 |
| 3 | 1.5 | 单赞价值+50%，转化权重加成 |
| 4 | 2.0 | 单赞价值翻倍，转化最高 3 倍封顶 |

### 6.3 奖励金权限

| 等级 | 权限 |
|------|------|
| 0 | 无法领取任何奖励金 |
| 1 | 仅基础奖励金 50%，无浮动/追加 |
| 2 | 全额基础+浮动+常规点赞追加，单笔 80% 封顶 |
| 3 | 所有上限+50%，可领内容转化追加，单笔 90% 封顶 |
| 4 | 所有上限+100%，内容转化无上限，优先结算、免冻结 |

### 6.4 功能与权益

| 等级 | 权益 |
|------|------|
| 0 | 仅浏览，无下单/评价/社群；点赞保持开放（后台权重 0，以《点赞追加奖金体系》为准） |
| 1 | 基础下单/评价/点赞，可进社群 |
| 2 | 完整功能，竞价、福利、先行赔付 |
| 3 | 专属标识、免费专家咨询、优先推送、商家折扣、维权优先 |
| 4 | KOC 标识、终身专家、专属客户经理、订单监理、年度福利、新功能内测、参与规则制定 |

---

## 七、通用核算规则

- **核算方式**：实时核算（无月度批量任务）
- **升级**：必须同时满足对应等级所有准入条件，不可跳级
- **降级**：未满足保级条件自动降级；严重违规可跳级降至 0
- **保级**：连续 3 个月无有效交易/有效评价，自动降 1 级，直至 1 级

---

## 八、与现有体系兼容

- **评价体系**：复用有效评价标准，将零散账号权重系统化为等级
- **点赞追加奖金**：直接替代「基础可信度权重」，公式不变
- **商户合规**：高等级评价/投诉权重更高
- **防刷**：本体系为核心防刷底层
- **竞价分发**：高等级竞价需求优先推送给优质商户
