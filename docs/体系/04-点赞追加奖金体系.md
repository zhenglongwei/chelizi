# 点赞追加奖金体系

> **定位**：长效激励的核心模块，由《03-全链路激励驱动体系》引用。锚定评价对其他车主的真实决策价值，实现「内容价值越高，收益越高，上不封顶」。与用户账号可信度等级、评价内容质量等级 100% 兼容。

---

## 一、底层原则

| 原则 | 说明 |
|------|------|
| 中立性优先 | 好评差评规则一致；激励锚定「内容对用户的决策价值」，而非「为门店引流」 |
| 价值与收益对等 | 内容价值、决策影响与奖金正相关，上不封顶 |
| 盈利绝对可控 | 常规≤80% 佣金；转化奖金从成交佣金单独列支，零垫资 |
| 防刷前置 | 权重分层、滞后结算、违规追溯 |
| 终身可追溯 | 全程留痕，违规一经核实即可追回全部奖金 |

---

## 二、核心术语

| 术语 | 定义 |
|------|------|
| 有效评价 | 交易闭环、凭证满足、有相关描述、账号合规；可参与点赞追加核算 |
| 有效点赞 | 停留≥15 秒、账号权重＞0、单用户单评价终身 1 次、无刷赞异常 |
| 账号综合权重系数 | 基础可信度权重 × 车型匹配权重（基础可信度见《01-用户账号可信度等级体系》） |
| 7 天决策窗口期 | 点赞后 7 日内用户成交，可触发该评价的内容转化奖金 |
| 内容转化奖金池 | 每笔成交佣金 50% 划出，仅分配给影响决策的有效评价发布者 |
| 事后验证有效点赞 | 下单前已浏览、修完车后回头点赞，适用专属加权与补发 |

---

## 三、三大奖金模块

### 3.1 常规点赞追加

```
单条评价常规点赞追加奖金 = 订单实收佣金 × 0.5% × 当月新增有效点赞的「账号综合权重系数」之和
```

**车型匹配权重**（账号综合权重系数 = 基础可信度权重 × 车型匹配权重）：

| 匹配等级 | 判定标准 | 权重 |
|----------|----------|------|
| 精准匹配 | 完全同款车型（同品牌、同车系、同年款）；实现上可简化为车牌号一致 | 2.0 |
| 高度匹配 | 同品牌、同车系不同配置/年款 | 1.5 |
| 基础匹配 | 同价位、同品类、同级别 | 1.0 |
| 无匹配 | 未完善车辆信息或完全不相关 | 0.5 |

**结算周期**：每月 10 日定期结算上月应得奖励，到账预留 1～3 个工作日复核。用户可查看详细清单，详见《评价与点赞奖励-定期结算方案》。

### 3.2 内容转化追加

```
单条评价可获得的单笔成交转化奖金 = 内容转化奖金池 ×（该评价决策权重 ÷ 本次成交所有有效评价决策权重总和）
```

**决策权重** = 决策时间权重 × 内容停留权重 × 内容匹配权重 × 内容价值权重（四维相乘）

**决策权重系数表**：

| 权重维度 | 等级与系数 |
|----------|------------|
| 决策时间（40%） | 下单前 24h 内点赞 4.0；24–72h 2.0；3–7 天 1.0 |
| 内容停留（30%） | ≥3 分钟 3.0；1–3 分钟 2.0；30 秒–1 分钟 1.0 |
| 内容匹配（20%） | 同车型+同项目 3.0；同车型+同类 2.0；同价位+同类 1.0 |
| 内容价值（10%） | 4 级爆款 3.0；3 级标杆 2.0；2 级优质 1.0；1 级基础 0.5 |

**准入**：有效评价 + 7 天窗口内有效点赞 + 车型/项目匹配 + 单笔最多 10 条分配。

### 3.3 事后验证有效点赞

**判定（5 项全满足）**：下单前 7 天有浏览 + 同品牌同车系同类项目真实交易 + 完工后 30 天内点赞 + 有效点赞标准 + 单笔交易最多 1 条触发。

**前台**：排序权重×2；≥10 次打「车主验证・真实避坑」红标；单独展示「XX 位车主验证有用」。

**后台**：常规点赞全额核算；该笔成交未给其他评价分配过时，从奖金池划出 50% 补发。

## 四、封顶与结算

| 规则 | 说明 |
|------|------|
| 可叠加 | 基础 + 追评差额补发 + 常规点赞追加 + 内容转化追加 + 事后验证补发 |
| 常规封顶 | 基础 + 追评差额补发 + 常规点赞追加 ≤ 实收佣金 80% |
| 超额无上限 | 内容转化追加、事后验证补发不纳入封顶 |
| 结算 | 基础奖励审核通过即可解冻提前提现；评价升级差额、点赞追加（含常规/内容转化/事后验证）每月 10 日定期结算，到账预留 1～3 个工作日，用户可查看详细清单 |

---

## 五、四维一体反作弊

| 维度 | 校验 |
|------|------|
| 设备 | 唯一性、黑产库、多账号；一设备一主账号 |
| 账号 | 实名、车辆、可信度、违规；一人一证一号 |
| 行为 | 浏览/点赞/评价/下单路径；秒赞、秒下单、批量操作拦截 |
| 交易 | 真实性、商户关联、支付异常；高频关联冻结核查 |
