# 全链路激励驱动体系

> **定位**：唯一核心作用是驱动用户持续产出真实、高价值评价，完全服务于「保障评价真实性」的核心目标，而非拉新补贴。100% 由底层评价体系驱动，无人工干预、付费干预空间。

---

## 一、设计原则

- **奖励仅与完整性、真实性绑定**：与评价正负倾向（好评/差评）完全无关，符合内容要求即全额发放
- **单订单奖励金 ≤ 税后佣金 70%**：从根源保证平台单订单边际利润为正
- **80% 客观可校验**：评价体系以客观内容为核心，20% 主观为补充，不搞「主观好评换奖励」
- **全流程资金闭环**：通过持牌第三方支付分账，平台不触碰维修资金、不形成资金池

---

## 二、基础激励

### 2.1 首次评价奖金

```
基础奖励 = 复杂度基础固定奖励 × 车价校准系数 × 维修复杂度校准系数
```

| 维修复杂度 | 基础固定奖励 |
|------------|--------------|
| L1 极低 | 10 元 |
| L2 低 | 30 元 |
| L3 中 | 100 元 |
| L4 高 | 300 元 |

**车价校准系数**：10 万及以下 1.0；10–20 万 1.2；20–30 万 1.5；30–50 万 2.0；50 万以上 3.0。

**维修复杂度校准系数**：基础项目 1.0；高难度项目 1.5；保险事故车项目 2.0。

**佣金比例**（用于计算实收佣金、70% 红线、80% 封顶）：

| 订单金额区间 | 基准佣金比例 |
|--------------|--------------|
| 5000 元以内 | 8% |
| 5000～20000 元 | 10% |
| 20000 元以上 | 12% |

注：可按服务商季度合规率/投诉率浮动（合规率≥95%、投诉率≤1% 下调 1%；合规率＜80% 或违规上调 2%），上下限为基准的 50%～120%。

**订单分级与封顶**：

| 订单分级 | 金额区间 | 单订单总奖励封顶 |
|----------|----------|------------------|
| 一级 | ≤1000 元 | 30 元 |
| 二级 | 1000～5000 元 | 150 元 |
| 三级 | 5000～20000 元 | 800 元 |
| 四级 | >20000 元 | 2000 元 |

按维修复杂度的单订单封顶：L1 50 元、L2 200 元、L3 800 元、L4 2000 元；保险事故车专属 3000 元。低端车型封顶可上浮 20%。

**双重约束**：取 min(按订单分级封顶, 按复杂度封顶, 实收佣金×70%)。

注：不设「选答 10% 加成」，完成必答题即全额发放基础奖励金。

### 2.2 追评奖金（整体评估 + 差额补发）

**核心原则**：主评价与追评视为**同一评价整体**，按「主评价 + 已有追评」整体评估内容质量等级；追评提交后若等级升级，按**差额补发**（升级后总奖金 − 已发奖金）。

| 机制 | 说明 |
|------|------|
| 主评价提交 | 按当时等级发放：基础奖励 + 浮动（1 级无、2 级 50%、3 级 100%） |
| 追评提交 | 触发整体重评估（AI 或规则）；若等级升级，补发差额 |
| 差额公式 | 补发金额 = 升级后总奖金（基础 + 新等级浮动）− 已发奖金 |
| 时间与次数 | **不限制**：用户可随时追加追评，次数不限；判定时结合质保期等综合评估，详见《02-评价内容质量等级体系》追评判定规则 |

**示例**：主评价 1 级已发 30 元 → 追评后整体升 2 级（应得 45 元）→ 补发 15 元。

**结算周期**：评价升级差额补发纳入每月定期结算，与点赞追加一并发放，详见《评价与点赞奖励-定期结算方案》。

### 2.3 优质内容浮动奖励

从佣金剩余 30% 中列支：

| 评价等级 | 浮动奖励 |
|----------|----------|
| 基础有效 | 无 |
| 优质高价值 | 基础奖励的 50% |
| 爆款标杆 | 基础奖励的 100% |

## 三、长效激励：点赞追加奖金

**核心逻辑**：锚定评价对其他车主的真实决策价值，实现「内容价值越高，收益越高，上不封顶」。

详见 **[04-点赞追加奖金体系](./04-点赞追加奖金体系.md)**，包含：

- **常规点赞追加**：基于内容认可度的基础追加
- **内容转化追加**：基于内容决策价值的超额追加
- **事后验证有效点赞**：修车后回头点赞的专属激励

**结算周期**：点赞追加奖金（含常规、内容转化、事后验证）每月 10 日定期结算，结算上月应得；到账预留 1～3 个工作日复核。用户可查看详细清单。

## 四、4 阶评价闭环（与奖励发放节点对应）

| 阶段 | 触发节点 | 奖励规则 |
|------|----------|----------|
| 第一阶：报价服务评价 | 选定服务商、确认维修前 | 选填，完成可叠加额外权益 |
| 第二阶：维修过程评价 | 维修完工、确认完工前 | 三/四级必填，一/二级选填 |
| 第三阶：完工验收评价 | 车辆完工、维修款结算后 | 全等级必填，核心奖励依据 |
| 第四阶：使用体验追评 | 完工后随时可追加 | 不限制时间与次数；追评提交后触发整体重评，若等级升级则差额补发；判定结合质保期等，详见 02 文档 |

---

## 五、合规红线（不可突破）

1. 奖励金仅与评价完整性、真实性绑定，与好评/差评无关
2. 单订单奖励金总支出（含个税）≤ 税后佣金 70%
3. 评价 80% 客观可校验，不搞「主观好评换奖励」
4. 全流程资金通过持牌支付分账闭环
