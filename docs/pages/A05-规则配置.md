# 规则配置页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | /admin/rules |
| 页面标识 | 规则配置、竞价规则 |
| 载体 | Web 管理后台 |
| 源码位置 | web/src/admin/pages/RuleConfig.tsx |

## 页面功能概述

配置平台**推荐与竞价规则**，如推荐权重、报价预警、偏差惩罚等。**奖励金规则、订单分级、佣金配置**已迁移至 [A10 奖励金规则配置](./A10-奖励金规则配置.md)。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 规则表单 | 各规则项的可编辑表单 |
| 保存按钮 | 提交保存 |

## 功能点详细说明

### 功能点 1：规则配置

| 项目 | 说明 |
|------|------|
| 功能 ID | F-A-RULE-001 |
| 功能描述 | 配置推荐权重、竞价、偏差等规则；奖励金规则见 A10 |
| 操作流程 | 修改配置项 → 保存 |
| 预期结果 | 写入 settings 表或配置存储 |
| 备注 | 具体规则项以产品设计为准 |

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/admin/config | GET | 获取配置 |
| /api/v1/admin/config | PUT | 更新单条配置 |
| /api/v1/admin/config/batch | POST | 批量更新配置 |

## 样式说明

后台为 Web 页面，样式独立于小程序。
