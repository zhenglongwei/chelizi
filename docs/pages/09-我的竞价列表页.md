# 我的竞价列表页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | `/pages/bidding/list` |
| 页面标识 | 我的竞价、竞价列表 |
| TabBar 位置 | 否 |

## 页面功能概述

展示用户发起的竞价记录，按状态筛选（进行中、已结束），支持查看竞价详情、报价列表，或继续发起新竞价。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 状态 Tab | 进行中、已结束 |
| 竞价列表 | 竞价卡片，支持分页/下拉刷新 |
| 竞价卡片 | 定损概要、竞价范围、倒计时/结束时间、报价数量 |
| 操作按钮 | 查看报价、结束竞价（进行中）、重新发起（已结束） |

## 功能点详细说明

### 功能点 1：竞价列表展示

| 项目 | 说明 |
|------|------|
| 功能 ID | F-BIDDING-LIST-001 |
| 功能描述 | 分页展示用户竞价 |
| 操作流程 | 进入页面或切换 Tab |
| 预期结果 | 展示竞价卡片；支持下拉刷新、上拉加载 |

### 功能点 2：状态筛选

| 项目 | 说明 |
|------|------|
| 功能 ID | F-BIDDING-LIST-002 |
| 功能描述 | 按进行中/已结束筛选 |
| 操作流程 | 点击 Tab |
| 预期结果 | 列表仅展示对应状态竞价 |

### 功能点 3：查看报价

| 项目 | 说明 |
|------|------|
| 功能 ID | F-BIDDING-LIST-003 |
| 功能描述 | 进入竞价报价页查看报价 |
| 操作流程 | 点击「查看报价」或卡片 |
| 预期结果 | 跳转 `/pages/bidding/detail?id={bidding_id}` |

### 功能点 4：结束竞价

| 项目 | 说明 |
|------|------|
| 功能 ID | F-BIDDING-LIST-004 |
| 功能描述 | 用户主动结束竞价（进行中） |
| 操作流程 | 点击结束竞价 → 确认 |
| 预期结果 | 竞价状态变为已结束；不再接收新报价 |

### 功能点 5：重新发起

| 项目 | 说明 |
|------|------|
| 功能 ID | F-BIDDING-LIST-005 |
| 功能描述 | 已结束未选厂的竞价可重新发起 |
| 操作流程 | 点击重新发起 |
| 预期结果 | 跳转 AI 定损页或复用原报告发起新竞价 |

## 页面参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| status | string | 否 | 初始筛选：ongoing、finished |

## 交互逻辑

1. **进入页面**：从个人中心跳转可带 status。
2. **切换 Tab**：更改 status，重新请求列表。
3. **进行中**：显示倒计时；可结束竞价。
4. **已结束**：显示结束时间；若已选厂则显示订单入口。

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/user/biddings | GET | 获取用户竞价列表，支持 status、page、limit |

## 异常与边界

- 未登录：跳转登录。
- 空列表：展示空状态，可引导发起竞价。
- 竞价已过期：定时任务关闭后，列表自动归入已结束。
