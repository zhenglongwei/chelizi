# 服务商工作台

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | `/pages/merchant/home/index` |
| 页面标识 | 服务商首页、工作台 |
| 载体 | 微信小程序 |
| 入口 | 服务商登录成功后默认进入 |

## 页面功能概述

服务商端首页，展示待报价数量、待接单数量、维修中订单数等汇总信息，提供竞价邀请、我的订单等快捷入口。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 顶部信息 | 维修厂名称、欢迎语 |
| 数据卡片 | 待报价、待接单、维修中、待确认 数量 |
| 快捷入口 | 竞价邀请、我的订单、维修厂信息 |
| 最近动态 | 可选，最新竞价/订单列表 |

## 功能点详细说明

### 功能点 1：数据汇总

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-HOME-001 |
| 功能描述 | 展示待报价、待接单、维修中、待确认数量 |
| 操作流程 | 进入页面自动加载 |
| 预期结果 | 数字准确，点击可跳转对应列表并筛选 |

### 功能点 2：竞价邀请

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-HOME-002 |
| 功能描述 | 跳转竞价邀请列表 |
| 操作流程 | 点击「竞价邀请」或待报价数字 |
| 预期结果 | 跳转 `/pages/merchant/bidding/list` |

### 功能点 3：我的订单

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-HOME-003 |
| 功能描述 | 跳转本店订单列表 |
| 操作流程 | 点击「我的订单」 |
| 预期结果 | 跳转 `/pages/merchant/order/list` |

### 功能点 4：维修厂信息

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-HOME-004 |
| 功能描述 | 跳转维修厂信息页 |
| 操作流程 | 点击「维修厂信息」 |
| 预期结果 | 跳转 `/pages/merchant/shop/profile` |

## 页面参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 无 | - | - | - |

## 交互逻辑

1. **进入页面**：需已登录，否则跳转登录。
2. **加载**：请求汇总数据，展示各数量。
3. **点击卡片/入口**：跳转对应页面，可带 status 参数筛选。

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/merchant/dashboard | GET | 获取工作台汇总数据，返回 { pending_bidding_count, pending_order_count, repairing_count, pending_confirm_count } |

## 异常与边界

- 未登录：跳转登录页。
- 数据为 0：正常展示 0，点击仍可进入列表页。
