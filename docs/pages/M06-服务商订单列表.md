# 服务商订单列表页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | `/pages/merchant/order/list/index` |
| 页面标识 | 我的订单、本店订单 |
| 载体 | 微信小程序 |
| 入口 | 服务商工作台「我的订单」 |

## 页面功能概述

展示本店维修订单列表，按状态筛选（待接单、维修中、待确认、已完成），支持查看详情、接单、更新状态。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 状态 Tab | 全部、待接单、维修中、待确认、已完成 |
| 订单列表 | 订单卡片，支持分页/下拉刷新 |
| 订单卡片 | 订单号、车辆、报价金额、状态、下单时间、**佣金比例、复杂度等级** |
| 操作 | 点击进入详情 |

## 功能点详细说明

### 功能点 1：订单列表展示

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-ORD-LIST-001 |
| 功能描述 | 分页展示本店订单 |
| 操作流程 | 进入页面或切换 Tab |
| 预期结果 | 展示订单卡片；支持下拉刷新、上拉加载 |
| 备注 | 仅展示 shop_id=本店的订单 |

### 功能点 2：状态筛选

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-ORD-LIST-002 |
| 功能描述 | 按状态筛选订单 |
| 状态说明 | 0 待接单、1 维修中、2 待确认、3 已完成 |
| 预期结果 | 切换 Tab 后列表仅展示对应状态 |

### 功能点 3：进入详情

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-ORD-LIST-003 |
| 功能描述 | 进入订单详情 |
| 操作流程 | 点击订单卡片 |
| 预期结果 | 跳转 `/pages/merchant/order/detail/index?id={order_id}` |

## 页面参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| status | string | 否 | 初始筛选：0、1、2、3 |

## 交互逻辑

1. **进入页面**：加载本店订单列表。
2. **切换 Tab**：更改 status，重新请求列表。
3. **点击卡片**：跳转订单详情。
4. **空列表**：展示「暂无订单」。

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/merchant/orders | GET | 获取本店订单列表，支持 status、page、limit，返回 { list, total } |

## 异常与边界

- 未登录：跳转登录。
- 订单被车主取消：状态变为已取消，可归入「全部」或单独 Tab。
