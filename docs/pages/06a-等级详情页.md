# 等级详情页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | `/pages/user/level-detail/index` |
| 页面标识 | 等级详情 |
| 入口 | 个人中心页 → 点击等级名称 |

## 页面功能概述

展示用户当前等级、权益、保级条件及升级进度，引导用户完成实名认证和车辆绑定。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 等级标识 | L0–L4 徽章、等级名称 |
| 当前权益 | 当前等级对应的功能与收益权限 |
| 保级条件 | 维持当前等级需满足的条件 |
| 升级进度 | 下一等级所需条件及当前达成情况 |

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/user/level-detail | GET | 获取等级详情（level、perks、retention、upgrade_progress） |

## 交互逻辑

1. 进入页面需登录，拉取等级详情。
2. 0 级用户展示「完成实名认证和车辆绑定」引导，可跳转完善资料页。
3. 支持下拉刷新。
