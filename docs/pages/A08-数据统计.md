# 数据统计页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | /admin/statistics |
| 页面标识 | 数据统计、数据分析 |
| 载体 | Web 管理后台 |
| 源码位置 | web/src/admin/pages/DataStatistics.tsx |

## 页面功能概述

展示平台业务数据统计，如订单量、GMV、维修厂分布、趋势图表等，支持按日期范围筛选。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 日期筛选 | 开始日期、结束日期 |
| 统计卡片 | 订单总数、GMV、维修厂数、用户数、**奖励金支出、分布**等 |
| 图表 | 订单趋势、维修厂分布、分类占比等 |

## 功能点详细说明

### 功能点 1：数据统计

| 项目 | 说明 |
|------|------|
| 功能 ID | F-A-STAT-001 |
| 功能描述 | 按日期范围展示统计数据 |
| 操作流程 | 选择日期范围 → 加载 |
| 预期结果 | 展示订单量、GMV、维修厂数等 |
| 备注 | 数据来源于 orders、shops、users 等表 |

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/admin/statistics | GET | 统计数据，支持 startDate、endDate |

## 样式说明

后台为 Web 页面，样式独立于小程序，可使用 ECharts 等图表库。
