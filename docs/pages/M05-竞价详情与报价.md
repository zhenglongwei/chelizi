# 竞价详情与报价页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | `/pages/merchant/bidding/detail/index` |
| 页面标识 | 竞价详情、提交报价 |
| 载体 | 微信小程序 |
| 入口 | 竞价邀请列表点击卡片 |

## 页面功能概述

服务商查看竞价详情（含 AI 定损报告、车辆信息、保险类型等），填写**预报价**明细（零配件、工艺、价格；**每项须填项目质保月数 `warranty_months`**）、工期后提交（**不设整单质保字段**，避免与分项歧义）。**评价从报价开始**：预报价与到店后最终报价将一并进入车主评价对比。支持 **Excel 模板下载（含中文使用说明）→ 填写保存 .xlsx → 小程序「导入 Excel」上传解析 → 预览确认**；支持 **自有报价单拍照/PDF → OCR/AI 结构化填单**，字段缺失则阻断并提示补充。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 定损报告卡片 | **事故照片**（定损依据，顶部展示，可点击预览）→ 车辆、支付方式（保险/自费）、保险时含事故类型/保险公司/对家保险/主责方、损伤部位（AI 分析参考）、复杂度等级（L1-L4） |
| 报价明细 | 每项含：损失部位、维修方案（换/修）+ 配件类型（换时四选一）；支持「采用 AI 建议」、新增/删除/编辑 |
| 增值服务 | 如代步车、上门接送等，每项一行，服务商自行填写 |
| 总报价 | 用户自行填写 |
| 工期、备注 | 工期（天）、备注（选填）；**项目质保仅行内 `warranty_months`** |
| 模板与导入 | 「下载 Excel 模板」（含中文说明与工作表「报价明细」）「导入 Excel」（.xlsx）「拍照识别报价单」；解析失败或缺字段时高亮错误 |
| 提交按钮 | 提交**预报价** |

## 功能点详细说明

### 功能点 1：查看定损报告

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-BID-DETAIL-001 |
| 功能描述 | 展示定损报告供报价参考 |
| 操作流程 | 进入页面自动加载 |
| 预期结果 | 顶部展示用户上传的事故照片（定损依据，可点击预览大图）；车辆、损伤部位（AI 分析参考）、维修建议等 |
| 备注 | 照片为定损依据，AI 分析仅供参考；数据来自 damage_reports |

### 功能点 2：提交报价

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-BID-DETAIL-002 |
| 功能描述 | 服务商提交报价（含明细，供后期评价对比） |
| 操作流程 | 可「采用 AI 建议」、**下载 Excel 模板**（电脑或手机用 Excel/WPS 打开，按说明填写保存 .xlsx 后导入）、**拍照识别报价单**（需配置千问 Key），或手动添加明细 → 填写总报价、工期、备注 → 提交 |
| 预期结果 | 提交成功，报价及 items 写入 quotes 表，提示「报价已提交」 |
| 限制 | 同一竞价同一店铺只能报价一次 |

### 功能点 3：联系车主

| 项目 | 说明 |
|------|------|
| 功能 ID | F-M-BID-DETAIL-003 |
| 功能描述 | 可选，展示车主联系方式或通过平台联系 |
| 备注 | 按产品规则，可能不直接展示车主手机号 |

## 页面参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | string | 是 | 竞价 ID（bidding_id） |

## 交互逻辑

1. **进入页面**：根据 id 加载竞价详情、定损报告。
2. **填写报价**：报价金额、预计工期必填；**每项须填项目质保（月）**；维修项目可预设或自定义。
3. **提交**：校验金额 > 0、工期 ≥ 0、分项质保 ≥ 0 → 调用报价接口 → 成功则提示并返回。

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/merchant/bidding/:id | GET | 获取竞价详情；响应扩展：`complexity_level`、`reward_tier`（订单分级） |
| /api/v1/merchant/quote | POST | 提交预报价，请求体 { bidding_id, amount, items, duration, remark, quote_import_source?, quote_import_file_url? }；**items 须含 `warranty_months`**（不设整单 `warranty`） |
| /api/v1/merchant/quote-template | GET | 返回 `filename`、`csv`、列说明 JSON（兼容旧版/脚本） |
| /api/v1/merchant/quote-template.xlsx | GET | 返回 **Excel 二进制**（`Content-Disposition: attachment`）；含工作表「使用说明」「报价明细」，表头与字段说明均为中文；小程序侧 `wx.request` + `writeFile` + `openDocument` |
| /api/v1/merchant/quote-import/preview | POST | Body `{ csv_text }` → `{ items, amount_sum }`；行级错误返回 400（脚本/兼容） |
| /api/v1/merchant/quote-import/preview-xlsx | POST | `multipart/form-data`，字段名 `file`，上传 **.xlsx**；解析工作表「报价明细」或首列表头含「损失部位」的表；返回结构与 CSV 预览一致；支持可选 AI 规范化（与 CSV 预览相同） |
| /api/v1/merchant/quote-sheet/analyze-image | POST | Body `{ image_url }`（须先 `merchant/upload/image`）；返回 `items`、`amount`/`duration`、`missing_fields`、`recognition_failed` |

## 异常与边界

- 竞价不存在或已结束：提示并返回。
- 本店已报价：提示「您已提交过报价」。
- 本店不在竞价范围内：提示「该竞价未邀请您」。
- 报价金额非法：提示「请输入有效金额」。
