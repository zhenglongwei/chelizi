# 个人中心页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | `/pages/user/index` |
| 页面标识 | 个人中心、我的 |
| TabBar 位置 | 是（底部 TabBar 最后一项「我的」；消息入口已独立为 Tab「消息」） |

## 页面功能概述

用户个人中心入口，展示头像、昵称、钱包、订单、竞价、收藏、浏览历史，以及设置等。**未登录时不请求、不展示**基于 openid 的服务商入口。用户在本页点击「登录」并完成车主授权后（或冷启动首次进入本页且本地已有车主 token），再调用 `/api/v1/merchant/check-openid` 判断当前微信是否已在 `merchant_users` 注册为服务商；若是，则展示「进入服务商工作台」（本地已有服务商 token 时直达；否则点击后通过微信快捷登录换 token）。未注册服务商的用户仍可通过设置中的服务商注册/登录进入。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 用户信息区 | 头像、昵称、等级标识（可点击进入等级详情页） |
| 我的钱包 | 奖励金余额、累计奖励金、提现按钮 |
| 我的订单 | 合并入口（`/pages/order/hub`），标题行右侧「预约」与维修厂详情预约逻辑一致（全平台可预约项）；快捷：全部、待评价、已完成 |
| 我的竞价 | 进行中、已结束 |
| 我的收藏 | 收藏的维修厂 |
| 浏览历史 | 最近浏览的维修厂 |
| 服务商工作台 | 绑定服务商微信时展示，可从此进入（无需先进设置） |
| 设置 | 账号设置、关于我们、联系客服、服务商注册/登录入口 |

## 功能点详细说明

### 功能点 1：奖励金提现

| 项目 | 说明 |
|------|------|
| 功能 ID | F-USER-001 |
| 功能描述 | 申请提现奖励金余额到微信零钱 |
| 操作流程 | 1. 点击提现 2. 输入金额（大于 0） 3. 选择提现方式（微信零钱） 4. 确认 5. 调用 /api/v1/user/withdraw |
| 预期结果 | 显示可提现余额；输入后显示预计到账时间；确认后提示「提现申请已提交」；1–3 个工作日到账 |
| 备注 | 单日最高 5000 元；记录提现历史 |

## 交互逻辑

1. **进入页面**：拉取用户信息等（需登录项按原逻辑）。**不在未登录时**检测服务商 openid。
2. **车主登录成功后**：用户在本页点击「登录」并授权通过返回本页，或冷启动首次展示且已有车主 token 时，执行一次 `wx.login` + `/api/v1/merchant/check-openid`；若已注册服务商则展示「进入服务商工作台」。切换 Tab 再次进入本页不重复检测（直至退出车主登录）。
3. **服务商工作台**：若本地已有 `merchant_token`，直达工作台；若无 token 但上一步已判定 openid 为服务商，点击后调用 `/api/v1/merchant/wechat-login` 换 token 再进入。
4. **提现**：点击提现 → 弹窗/新页输入金额 → 调用 withdraw 接口 → 提示结果。
5. **订单/竞价**：点击 → 跳转列表页并带 status。
6. **设置**：按模块分别跳转到对应页面。

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/user/profile | GET | 获取用户信息（含 level、level_name、needs_verification） |
| /api/v1/user/level-detail | GET | 获取等级详情（权益、保级、升级进度） |
| /api/v1/user/balance | GET | 获取余额/统计 |
| /api/v1/user/withdraw | POST | 申请提现 |
| /api/v1/user/orders | GET | 订单列表（或仅统计） |
| /api/v1/user/biddings | GET | 竞价列表（或仅统计） |
| /api/v1/merchant/check-openid | POST | 检测当前微信是否绑定服务商（`{ code }`），用于展示入口 |
| /api/v1/merchant/wechat-login | POST | 无本地服务商 token 时，点击入口后换 token（`{ code }`） |

## 未登录处理

- 未登录时展示登录引导，部分功能（钱包、订单等）需登录后可用。
- 点击需登录的功能时，先跳转登录/授权页。
