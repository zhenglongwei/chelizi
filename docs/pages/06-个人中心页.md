# 个人中心页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | `/pages/user/index` |
| 页面标识 | 个人中心、我的 |
| TabBar 位置 | 是（底部 TabBar 第 4 项「我的」） |

## 页面功能概述

用户个人中心入口，展示头像、昵称、钱包、订单、竞价、收藏、浏览历史，以及设置、服务商入口等。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 用户信息区 | 头像、昵称、会员等级 |
| 我的钱包 | 奖励金余额、累计奖励金、提现按钮 |
| 我的订单 | 全部订单、待评价、已完成 |
| 我的竞价 | 进行中、已结束 |
| 我的收藏 | 收藏的维修厂 |
| 浏览历史 | 最近浏览的维修厂 |
| 设置 | 账号设置、关于我们、联系客服、服务商注册/登录入口 |

## 功能点详细说明

### 功能点 1：奖励金提现

| 项目 | 说明 |
|------|------|
| 功能 ID | F-USER-001 |
| 功能描述 | 申请提现奖励金余额到微信零钱 |
| 操作流程 | 1. 点击提现 2. 输入金额（最低 10 元） 3. 选择提现方式（微信零钱） 4. 确认 5. 调用 /api/v1/user/withdraw |
| 预期结果 | 显示可提现余额；输入后显示预计到账时间；确认后提示「提现申请已提交」；1–3 个工作日到账 |
| 备注 | 最低 10 元；单日最高 5000 元；记录提现历史 |

### 功能点 2：我的订单

| 项目 | 说明 |
|------|------|
| 功能 ID | F-USER-002 |
| 功能描述 | 查看订单列表 |
| 操作流程 | 点击全部订单/待评价/已完成 |
| 预期结果 | 跳转 `/pages/order/list`，附带 status 参数 |
| 状态说明 | 待评价：维修完成未评价；已完成：已评价或已取消 |

### 功能点 3：我的竞价

| 项目 | 说明 |
|------|------|
| 功能 ID | F-USER-003 |
| 功能描述 | 查看竞价列表 |
| 操作流程 | 点击进行中/已结束 |
| 预期结果 | 跳转 `/pages/bidding/list`，附带 status 参数 |

### 功能点 4：我的收藏 / 浏览历史

| 项目 | 说明 |
|------|------|
| 功能 ID | F-USER-004 |
| 功能描述 | 查看收藏与浏览历史 |
| 操作流程 | 点击对应入口 |
| 预期结果 | 跳转维修厂列表页，预填收藏/历史筛选 |

### 功能点 5：设置入口

| 项目 | 说明 |
|------|------|
| 功能 ID | F-USER-005 |
| 功能描述 | 进入账号设置、关于我们、联系客服、服务商注册/登录 |
| 操作流程 | 点击对应项 |
| 预期结果 | 账号设置 → 设置页；关于/客服 → 对应页面；服务商 → 服务商注册/登录页 |

## 页面参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 无 | - | - | 作为 TabBar 页，通常无参数 |

## 交互逻辑

1. **进入页面**：需登录；拉取用户信息、余额、订单/竞价数量（可仅统计）。
2. **提现**：点击提现 → 弹窗/新页输入金额 → 调用 withdraw 接口 → 提示结果。
3. **订单/竞价**：点击 → 跳转列表页并带 status。
4. **设置**：按模块分别跳转到对应页面。

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/user/profile | GET | 获取用户信息 |
| /api/v1/user/balance | GET | 获取余额/统计 |
| /api/v1/user/withdraw | POST | 申请提现 |
| /api/v1/user/orders | GET | 订单列表（或仅统计） |
| /api/v1/user/biddings | GET | 竞价列表（或仅统计） |

## 未登录处理

- 未登录时展示登录引导，部分功能（钱包、订单等）需登录后可用。
- 点击需登录的功能时，先跳转登录/授权页。
