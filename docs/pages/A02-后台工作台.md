# 后台工作台

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | /admin/dashboard |
| 页面标识 | 工作台、数据概览 |
| 载体 | Web 管理后台 |
| 源码位置 | web/src/admin/pages/Dashboard.tsx |

## 页面功能概述

管理后台首页，展示平台核心数据概览，如订单数、维修厂数、今日数据等，便于运营快速了解业务状态。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 数据卡片 | 订单总数、维修厂数、今日订单、今日 GMV 等 |
| 图表 | 可选，订单趋势、维修厂分布等 |
| 快捷入口 | 订单管理、维修厂管理等 |

## 功能点详细说明

### 功能点 1：数据概览

| 项目 | 说明 |
|------|------|
| 功能 ID | F-A-DASH-001 |
| 功能描述 | 展示核心业务数据 |
| 操作流程 | 进入页面自动加载 |
| 预期结果 | 数字准确，可点击跳转详情页 |

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/admin/statistics | GET | 获取统计数据，支持 startDate、endDate |

## 样式说明

后台为 Web 页面，样式独立于小程序，使用 Ant Design。
