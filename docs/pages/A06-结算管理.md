# 结算管理页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | /admin/settlement |
| 页面标识 | 结算管理、佣金结算 |
| 载体 | Web 管理后台 |
| 源码位置 | web/src/admin/pages/SettlementManagement.tsx |

## 页面功能概述

展示订单结算、佣金数据，支持按维修厂、日期筛选，用于财务对账、提现审核等。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 筛选区 | 维修厂、日期范围 |
| 列表表格 | 订单号、维修厂、订单金额、佣金、**奖励金**、结算时间 |
| 汇总 | 总佣金、可提现金额等 |

## 功能点详细说明

### 功能点 1：结算列表

| 项目 | 说明 |
|------|------|
| 功能 ID | F-A-SETTLE-001 |
| 功能描述 | 展示已完成订单的结算数据 |
| 操作流程 | 进入页面或切换筛选 |
| 预期结果 | 展示订单、佣金、结算时间 |
| 备注 | 佣金按车型分级、复杂度分级；奖励金需区分个税处理（≤800 无税、>800 平台承担） |

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/admin/settlements | GET | 结算列表，支持筛选参数 |

## 样式说明

后台为 Web 页面，样式独立于小程序。
