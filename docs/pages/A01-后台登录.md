# 后台登录页

## 基本信息

| 项目 | 说明 |
|------|------|
| 页面路径 | /admin/login |
| 页面标识 | 管理后台登录、运维登录 |
| 载体 | Web 管理后台（React + Ant Design） |
| 源码位置 | web/src/admin/pages/Login.tsx |

## 页面功能概述

平台运营人员使用账号密码登录管理后台，登录成功后进入工作台，进行维修厂审核、订单监控、配置等操作。

## 页面布局结构

| 模块 | 说明 |
|------|------|
| 登录表单 | 用户名、密码 |
| 登录按钮 | 提交登录 |
| 品牌标识 | 车厘子 / 管理后台 |

## 功能点详细说明

### 功能点 1：账号密码登录

| 项目 | 说明 |
|------|------|
| 功能 ID | F-A-LOGIN-001 |
| 功能描述 | 管理员使用用户名+密码登录 |
| 操作流程 | 输入用户名、密码 → 点击登录 |
| 预期结果 | 登录成功，存储 admin_token，跳转工作台 |
| 备注 | 默认账号 admin / admin123（可配置） |

## 页面参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 无 | - | - | - |

## 交互逻辑

1. **进入页面**：未登录访问 /admin/* 时重定向至此。
2. **登录**：校验非空 → 调用登录接口 → 成功则存储 token，跳转 /admin/dashboard。
3. **失败**：展示错误信息。

## 依赖接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/admin/login | POST | 请求体 { username, password }，返回 { token, user } |

## 异常与边界

- 用户名或密码错误：提示「用户名或密码错误」。
- 401 响应：清除本地 token，跳转登录页。

## 样式说明

后台为 Web 页面，样式独立于小程序，使用 Ant Design 组件库，不参考《设计规范.md》。
