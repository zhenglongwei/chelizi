## 四、页面分类与业务流

### 车主端 TabBar

- 首页、服务商（维修厂搜索）、定损（居中）、消息、我的

### 核心业务流（三端协同）

1. **AI 定损 → 竞价 → 选厂 → 订单**：车主定损 → 发起竞价 → 服务商报价 → 车主选厂 → 订单生成
2. **维修完成**：服务商接单 → 维修中 → 待确认 → 车主确认完工 → 主评价页弹出（3 模块 1 次提交）→ 1 个月/3 个月追评弹窗（返厂评价等同）；是否需等分账完成由后台配置
3. **后台运维**：维修厂审核、订单监控、配置、统计

### 入口关系

- 车主端：个人中心 → 设置 → 服务商注册 / 服务商登录（直接跳 merchant/register、merchant/login）
- 服务商入口页：`/pages/merchant/index` 提供登录、注册、工作台三个入口（当前设置页未链至此，可考虑增加）
- 服务商首页：`/pages/merchant/home`（M03 工作台，登录后进入）
- 后台：`https://域名/admin/login`
