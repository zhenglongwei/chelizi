# 分项质保 — 协议引用稿与客服话术（原「电子质保凭证」）

> **非法律意见**：以下供产品纳入《用户服务协议》《服务商入驻协议》或订单说明时由法务修订后使用。产品已不以独立电子质保卡、防伪核验为主叙事。

## 一、建议条款要点（车主端）

1. **质保义务主体**：分项质保及维修质量责任由**接单维修厂（服务商）**向车主承担，以双方订单及已确认的维修/报价方案（含各项目质保月数）为准。  
2. **记录形式**：辙见在订单中展示已确认的方案与分项质保约定，**不提供**须盖章方有效的独立质保卡或平台「承保」承诺；不构成对维修质量、配件真实性或质保履行的**额外担保或保证**。  
3. **争议处理**：质保履行、维修质量争议应**优先联系维修厂**；辙见可依据规则提供订单信息协助协查，但不替代行政、司法或第三方鉴定结论。  
4. **补充材料**：历史订单中若存在商户上传的纸质质保类照片，仅作**补充参考**，以订单文字约定与司法/行政认定为准（具体由法务表述）。

## 二、建议条款要点（商户端）

1. 商户须按平台规则在方案中如实填写分项质保月数；商户对向车主作出的质保承诺**自行承担**相应责任。  
2. 辙见提供订单记录与展示，旨在**降低纠纷沟通成本**，**不转移**商户法定或约定义务。

## 三、客服话术（对齐）

- **用户问「这是平台保修吗？」**  
  → 「不是。质保是维修厂给您的承诺，**以您订单里确认的维修方案和分项质保月数为准**，您可以在订单详情里查看。」

- **用户问「出了问题找谁？」**  
  → 「**先联系接单维修厂**处理质保。如需，我们可以在规则范围内帮您核对订单信息。」

- **用户问「有没有电子质保卡/防伪码？」**  
  → 「我们**不再提供**单独质保卡或核验码；请以订单内的分项质保约定为准，建议保存好订单与结算凭证。」
