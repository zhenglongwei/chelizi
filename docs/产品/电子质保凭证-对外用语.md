# 分项质保 — 对外用语口径（原「电子质保凭证」）

> 产品已取消独立电子质保卡、卡面样式选择与「质保卡核验」；对外统一以**订单内维修/报价明细的分项质保（月数）**为准。与客服、协议引用时避免再承诺「平台出证/防伪核验」。

## 角色定义

| 角色 | 对外说法 | 避免 |
|------|----------|------|
| 维修厂 | **质保义务主体**、接单维修厂、本单服务商 | 含糊为「平台店」 |
| 辙见 | **订单记录与展示**、协助协查订单信息 | 「平台承保」「平台保修」「平台盖章质保卡」 |
| 约定依据 | **订单方案/报价明细**、各项目「质保 X 月」 | 单独使用「电子质保卡」「存证防伪码」作主叙事 |

## 常用说法

| 不宜 | 建议 |
|------|------|
| 电子质保凭证 / 存证防伪码 | 订单里的分项质保约定；以订单详情方案为准 |
| 预览/查看电子质保凭证 | 在订单详情查看维修方案与分项质保月数 |
| 质保卡核验 | （已下线）改为引导查看订单约定或联系维修厂 |

## 实现与文档同步检查

- 小程序：`pages/order/detail`、`pages/merchant/order/detail`、`pages/review/submit`、`pages/user/index`、`pages/help/warranty-notice`；历史页面与组件 `pages/order/warranty-card`、`pages/merchant/warranty-style`、`components/warranty-card-surface` 若仍存在于仓库，**不应再注册为小程序页面或入口**。
- 评价证据：`web/api-server/utils/review-for-review-evidence.js`（分项列表 + 可选历史补充图）
