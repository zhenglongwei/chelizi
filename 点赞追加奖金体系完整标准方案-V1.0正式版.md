## 一、方案核心底层原则（不可突破的规则根基）

| 原则 | 说明 |
|------|------|
| **中立性优先** | 好评与差评适用完全一致的奖金规则；彻底解绑评价所属门店与奖金收益，激励锚定「内容对用户的决策价值」，而非「为门店引流的效果」 |
| **价值与收益对等** | 评价内容的真实参考价值、对用户决策的影响程度，与奖金完全正相关，价值越高收益越高，上不封顶 |
| **盈利绝对可控** | 奖金从实收佣金列支；单笔基础奖金≤30% 佣金，常规总奖金≤80% 佣金；转化类超额奖金从新增成交佣金单独列支，零垫资、零亏损 |
| **全体系一致性** | 不推翻原有规则，仅增量延伸，复用复杂度分级、有效评价标准、商户资质、账号权重 |
| **防刷前置** | 权重分层、滞后结算、违规追溯，刷量成本远高于收益 |
| **终身可追溯** | 所有评价、点赞、交易行为全程留痕不可篡改，违规行为终身追溯，一经核实即可追回全部对应奖金、执行对应处罚 |

## 二、核心术语定义

**有效评价**：仅满足以下全部条件的评价，可参与点赞追加奖金核算；无效评价仅可被点赞、不产生奖金：

- 完成平台全交易闭环，系统核验订单真实有效、无刷单/合单/拆单违规
- 满足对应复杂度必传凭证（L1-L2 至少 1 张施工/新旧件实拍图，L3-L4 至少 2 张核心凭证图 + 维修明细单）
- 评价包含与维修项目相关的有效描述，无盗图、AI 生成、虚假、辱骂内容
- 发布账号非高风险、非封禁，无历史刷评/刷赞违规

**有效点赞**：前台 100% 计入点赞数；仅满足以下条件的点赞纳入后台奖金核算：

- 完整浏览评价页面，停留时长≥15 秒
- 点赞账号基础可信度权重＞0
- 单用户对同一条评价终身仅 1 次有效点赞，同设备/IP/手机号关联多账号仅 1 次有效
- 无集中刷赞、批量操作、IP 异常等违规

**账号综合权重系数**：

```
账号综合权重系数 = 基础可信度权重 × 车型匹配权重
```

**7 天决策窗口期**：用户对某条评价发起有效点赞后的 7 个自然日内，该周期内用户完成的平台成交，可触发该评价的内容转化奖金。

**内容转化奖金池**：每笔真实成交，从实收佣金中划出 50%，作为该笔成交专属的内容转化奖金池，仅分配给对该用户决策产生影响的有效评价发布者，与基础奖金、常规点赞追加完全独立。

**事后验证有效点赞**：用户下单前已完整浏览对应评价，完成维修交易后，回头对该评价发起的有效点赞，是对内容价值的终极验证，适用专属特殊加权与奖金规则，需满足本方案第五章的全部判定标准。

**内容转化效率分**：衡量评价内容真实决策价值的核心指标，作为评价排序、奖金核算的核心补充依据：

```
内容转化效率分 = （看过该评价后 7 天内下单的用户数 ÷ 该评价总浏览用户数）× 100% × 同项目匹配系数
```

注：仅同车型、同维修项目的下单行为计入核算；1 次事后验证有效点赞等同于 1 次有效转化。

## 三、前置固定规则（完全沿用原有体系）

1. **汽修项目复杂度四级分级标准**：完全沿用平台既定标准，L1-L4 等级划分、对应资质要求、凭证要求、评价权重完全不变。

2. **基础奖励体系**：

```
首次评价基础奖励 = 复杂度基础固定奖励 × 车价校准系数 × 多项目叠加校准系数 × 客单价超额校准系数
```

- 复杂度基础固定奖励：L1=10 元、L2=30 元、L3=100 元、L4=300 元，层级优先级永久锁定
- 追评奖励：7 天追评发放首次基础奖励的 20%，30 天追评发放首次基础奖励的 30%
- 基础奖励红线：单笔订单「首次评价 + 追评」基础奖励总额 ≤ 实收佣金 30%

3. **永久盈利生死红线**：

- 常规总奖金封顶：单笔订单「基础奖励 + 追评奖励 + 常规点赞追加奖金」总额 ≤ 实收佣金 80%
- 超额奖金安全垫：内容转化奖金池从新增成交佣金单独列支，每笔划出 50%，平台永久保留至少 50% 毛利

4. **用户账号可信度等级体系**：完全沿用平台 0-4 级账号分级规则，账号等级直接决定基础可信度权重、权限边界、收益上限。

## 四、核心主体：点赞追加奖金完整规则

点赞追加奖金分为**三大独立模块**，可叠加发放、互不冲突、独立核算，覆盖「内容本身认可度价值、内容对用户的决策价值、内容的终极验证价值」三大核心维度。

### 模块一：常规点赞追加奖金（基于内容认可度的基础追加）

**核心计算公式**：

```
单条评价当月常规点赞追加奖金 = 订单实收佣金 × 0.5% × 当月新增有效点赞的「账号综合权重系数」之和
```

**维度明细规则**：

| 维度 | 规则 |
|------|------|
| 单赞基础锚定值 | 订单实收佣金的 0.5%，全平台统一；高客单价、高复杂度订单单赞价值更高 |
| 基础可信度权重 | 见下表 |
| 车型匹配权重 | 见下表 |

**基础可信度权重**：

| 账号等级 | 判定标准 | 权重 |
|----------|----------|------|
| 4 级 核心可信用户 | 注册≥90 天，有效评价≥10 条，终身无违规 | 2.0 |
| 3 级 活跃可信用户 | 注册≥30 天，有效评价≥3 条，无违规 | 1.0 |
| 2 级 普通可信用户 | 注册≥7 天，≥1 笔真实成交，无违规 | 0.5 |
| 0-1 级 高风险/无效账号 | 注册＜7 天无交易、有违规、封禁 | 0 |

**车型匹配权重**：

| 匹配等级 | 判定标准 | 权重 |
|----------|----------|------|
| 精准匹配 | 完全同款车型（同品牌、同车系、同年款） | 2.0 |
| 高度匹配 | 同品牌、同车系不同配置/年款 | 1.5 |
| 基础匹配 | 同价位、同品类、同级别 | 1.0 |
| 无匹配 | 未完善车辆信息或完全不相关 | 0.5 |

**核算边界规则**：

- 核算周期：按月核算，每月 1 日统计上月（**实施已取消**：月度核算任务已取消，发放方式待定）
- 收益有效期：2 年；3 级及以上标杆/爆款评价可通过更新追评延长 1 年
- 无效追溯：点赞账号后续违规则追溯扣除

### 模块二：内容转化追加奖金（基于内容决策价值的超额追加）

**核心分配公式**：

```
单条评价可获得的单笔成交转化奖金 = 内容转化奖金池 ×（该评价决策权重 ÷ 本次成交所有有效评价决策权重总和）
```

**准入条件**：有效评价 + 7 天窗口期内有效点赞 + 车型/项目匹配 + 单笔成交最多 10 条评价分配。

**决策权重公式**：

```
决策权重 = 决策时间权重 × 内容停留权重 × 内容匹配权重 × 内容本身价值权重
```

| 权重维度 | 等级与系数 |
|----------|------------|
| 决策时间（40%） | 下单前 24h 内点赞 4.0；24-72h 2.0；3-7 天 1.0 |
| 内容停留（30%） | ≥3 分钟 3.0；1-3 分钟 2.0；30 秒-1 分钟 1.0 |
| 内容匹配（20%） | 同车型+同项目 3.0；同车型+同类 2.0；同价位+同类 1.0 |
| 内容价值（10%） | 4 级爆款 3.0；3 级标杆 2.0；2 级优质 1.0；1 级基础 0.5 |

**内容质量等级**：

| 等级 | 达标标准 |
|------|----------|
| 1 级 基础 | 有效评价基础要求 + 实拍凭证 + 相关描述 |
| 2 级 优质 | 1 级 + 维修明细单、新旧件对比、花费/避坑、7 天追评 |
| 3 级 标杆 | 2 级 + 施工节点图、检测报告、防伪记录、报价对比、30 天二次追评 |
| 4 级 爆款 | 3 级 + 同车型同项目≤10 条、7 天内同车型浏览≥1000、点赞≥50、人工复核 |

**核算边界规则**：

- 核算周期：按单笔成交实时核算，确认完工、无售后纠纷、7 天冻结期结束后自动核算
- 收益上限：奖金池 100% 全部分配，无上限
- 无效追溯：成交被判定虚假则作废并追溯扣除

### 模块三：事后验证有效点赞特殊规则（基于内容终极验证的专属激励）

本模块为纯增量优化，完全兼容前两大模块，针对「用户修车后回头点赞」设置专属加权与奖金规则。

**判定标准（5 项全满足）**：

| 门槛 | 要求 |
|------|------|
| 浏览决策 | 下单前 7 天内有该评价完整浏览记录（停留≥30 秒） |
| 交易匹配 | 完成同品牌同车系 + 同类维修项目的真实交易 |
| 时间节点 | 点赞在确认完工后 30 天内 |
| 有效点赞 | 符合平台有效点赞标准，单用户单评价终身仅 1 次 |
| 唯一性 | 同一笔交易最多 1 条评价触发特殊规则 |

**前台内容权重特殊加权**：

- 排序权重×2，计入内容价值分与内容转化效率分
- 累计≥10 次打「车主验证・真实避坑」专属红标，置顶展示
- 单独展示「XX 位车主验证有用」
- 发布者账号成长分额外加成

**后台奖金核算**：

- 常规点赞追加：全额正常核算
- 内容转化专属补发：
  - 触发条件：满足全部判定标准，且该笔成交未给其他评价分配过转化奖金
  - 补发标准：从奖金池划出 50% 一次性补发，剩余 50% 作废
  - 盈利兜底：平台保留至少 75% 佣金毛利；刷手套利投产比为负

## 五、奖金叠加、封顶与个税规则

- **可叠加**：基础奖励 + 追评 + 常规点赞追加 + 内容转化追加 + 事后验证专属补发，无互斥
- **常规封顶**：基础 + 追评 + 常规点赞追加 ≤ 实收佣金 80%
- **超额无上限**：内容转化追加、事后验证专属补发不纳入封顶，仅受奖金池限制
- **个税**：联合持牌支付机构代扣代缴，提供完税证明，用户到手为税后金额

## 六、全链路防刷与违规处罚规则

### 1. 四维一体反作弊基建（事前免疫）

四个维度交叉验证，任一维度异常直接触发风控：

| 维度 | 校验内容 |
|------|----------|
| 设备 | 设备唯一性、黑产库、多账号切换；一设备一主账号 |
| 账号 | 实名、车辆绑定、可信度等级、违规记录；一人一证一号 |
| 行为 | 浏览/点赞/评价/下单路径是否符合真实车主画像；秒赞、秒下单、批量操作直接拦截 |
| 交易 | 交易真实性、商户与用户关联、支付账户异常；高频关联交易冻结核查 |

### 2. 违规行为判定标准

| 违规类型 | 判定标准 |
|----------|----------|
| 虚假评价/刷评 | 伪造交易、盗图/PS、AI 生成、与门店合谋虚假好评/恶意差评 |
| 刷赞套利 | 批量小号、同设备/IP 多账号点赞、水军/工作室、机器点赞 |
| 虚假成交刷奖 | 虚假下单、拆单/合单套利、自买自卖套取转化奖金 |
| 恶意违规 | 辱骂/诽谤、盗用内容、恶意举报、批量扰乱 |

### 3. 阶梯式处罚措施（终身可追溯）

| 等级 | 措施 |
|------|------|
| 轻度 | 警告、取消违规内容奖金、账号权重下调 30 天 |
| 中度 | 追回违规奖金、权重永久清零、限制权限 90 天 |
| 重度 | 追回全部奖金、永久封禁、黑名单、身份证/手机/设备终身禁注册 |
| 涉嫌违法 | 移交公安机关、追究法律责任 |

## 七、结算与发放规则

- **实时反馈**：账号等级、评价权重、点赞奖金实时自动化展示；发布评价后可实时查看基础奖励、预计追加奖金、成长进度；每收到有效点赞实时更新奖金金额
- **结算周期**：每月 5 日前完成上月全量核算（**常规点赞追加月度核算已取消**，发放方式待定）
- **发放时间**：每月 10 日前完成税后发放；基础奖励审核通过后即可解冻提前提现
- **明细公示**：用户可查点赞明细、权重系数、成交分配、个税扣缴
- **异议处理**：结算后 7 个工作日内申请复核，3 个工作日内反馈
