一、先明确：国家法定维修资质的官方定义与经营范围（所有对应关系的法律根基）
以下内容全部来自 2021 年交通运输部修订的《机动车维修管理规定》，是全国通用的法定标准，也是汽修行业的准入红线，平台所有规则必须严格贴合，无任何变通空间。
法定资质等级	官方定义	法定可承接的经营范围（核心）	核心能力边界
一类机动车维修企业	从事机动车整车修理、总成修理、整车维护、小修、维修救援、专项修理和维修竣工检验工作的企业	1. 机动车整车大修、总成（发动机 / 变速箱 / 驱动电机等）拆解大修；

3. 整车维护、小修、专项维修、维修竣工检验全品类业务；
4. 新能源汽车三电系统总成大修、整车修复	全品类、全难度维修项目，是汽修行业最高资质，拥有整车大修、结构件修复、总成大修的独家法定经营权
二类机动车维修企业	从事机动车整车维护、小修、维修救援、专项修理工作的企业	1. 机动车常规保养、易损件更换、常规故障维修；
2. 核心部件的局部维修（不含总成拆解大修）；

关键法律风险提示：超范围经营的商户，将被处以 2000 元 - 3 万元的行政处罚；若因超范围维修导致车辆损坏、人员伤亡等事故，商户承担主要责任，平台明知商户超范围经营仍提供撮合服务的，将承担连带赔偿责任，情节严重的将被吊销平台相关经营资质。
二、L1-L4 复杂度分级与资质等级的法定对应标准
完全贴合上述法规要求，同时 100% 兼容你之前敲定的评价权重、奖励金、竞价分发、排序全体系，无任何冲突。每一级别明确「法定最低资质要求」和「平台落地补充规则」，无模糊地带，系统可直接自动拦截。
L1 极低复杂度（基础便民服务）
法定最低资质要求：三类机动车维修企业（对应专项资质）
可承接的资质范围：
拥有对应项目专项资质的三类维修企业（如洗车美容专项、轮胎修补专项、空调维修专项等）；
二类、一类维修企业，可全量承接所有 L1 项目。
平台落地规则
商户入驻时，仅上传对应专项资质，即可开通 L1 项目的接单、竞价权限；
无任何专项资质的商户，永久无法承接 L1 及以上任何项目，系统自动过滤；
与原有体系打通：L1 项目的评价权重固定为 0.2，不影响店铺核心评分，从根源杜绝刷分。
L2 低复杂度（常规维保 / 易损件更换）
法定最低资质要求：二类机动车维修企业，或拥有对应专项资质的三类维修企业
可承接的资质范围：
二类、一类维修企业，可全量承接所有 L2 项目；
三类维修企业，仅可承接与自身专项资质完全匹配的 L2 项目（例：拥有钣金喷漆专项资质的三类商户，可承接 L2 级单块钣金喷漆项目；拥有轮胎专项资质的三类商户，可承接 L2 级轮胎更换、四轮定位项目），禁止跨专项承接 L2 项目。
平台落地规则
系统自动匹配商户专项资质与订单项目，无对应资质的商户，无法看到、无法报名对应 L2 订单；
三类专项资质商户，仅能开通对应专项的 L2 接单权限，无法开通全品类 L2 权限；
与原有体系打通：L2 项目评价权重固定为 1.0，是平台评分的基准线，常规维保订单的评价不具备大幅拉高店铺评分的能力，杜绝刷分。
L3 中复杂度（常规故障维修 / 专项整备）
法定最低资质要求：二类及以上机动车维修企业，三类维修企业绝对不可承接
可承接的资质范围：
仅二类、一类维修企业可承接，三类维修企业无论拥有任何专项资质，均不可承接 L3 项目。
法规依据：L3 项目涉及车辆核心系统的拆解维修、多系统联动故障排查，超出了三类专项维修的法定经营范围，属于二类及以上资质的「整车维修、总成局部维修」范畴
平台落地规则
无二类及以上资质的商户，系统直接过滤所有 L3 订单，无任何接单、竞价权限；
二类资质商户承接 L3 项目，需额外上传对应项目的持证技师证书（中级及以上汽车维修工），无证书的商户关闭对应权限；
与原有体系打通：L3 项目评价权重固定为 3.0，是店铺评分的核心影响因子，仅真实的中高难度维修订单，才能真正拉高店铺评分，彻底堵死低难度订单刷分的路径。
L4 高复杂度（总成大修 / 整车修复 / 疑难故障）
法定最低资质要求：一类机动车维修企业，二类及以下资质绝对不可承接
可承接的资质范围：
仅拥有一类机动车维修资质的企业可承接，二类、三类资质无论规模多大、技术多强，均不可承接 L4 项目，无任何例外。
法规依据：L4 项目的核心是「发动机 / 变速箱 / 三电系统总成拆解大修、事故车车身结构件修复、整车修复」，这些项目全部属于一类维修资质的*独家法定经营范围，二类及以下资质承接属于明确的超范围违法经营*
平台落地规则
无一类维修资质的商户，系统永久屏蔽所有 L4 订单，无任何接单、竞价权限，无人工开通通道；
一类资质商户承接 L4 项目，需额外上传对应项目的高级技师证书、专项维修设备清单，无对应资质的关闭接单权限；
与原有体系打通：L4 项目评价权重固定为 6.0（保险事故车订单权重 ×2），是店铺评分、竞价梯队划分的最高权重因子，仅 L4 订单的真实评价，能决定店铺的顶级流量优先级；
额外公示要求：承接 L4 订单的商户，必须在店铺主页、报价页，醒目公示一类维修资质证书、对应项目技师资质，接受全平台用户监督。
三、平台全链路落地执行规则（与原有体系 100% 打通）
1. 入驻环节：资质前置核验，权限自动匹配
商户入驻时，必须上传交通运输部门颁发的《机动车维修经营许可证》，系统自动识别资质等级、经营范围、有效期，无有效资质的商户无法完成入驻；
系统根据资质等级，自动给商户开通对应复杂度等级的接单、竞价权限，超范围订单商户完全不可见，从根源杜绝超范围经营。
2. 竞价分发环节：资质为第一过滤门槛
你之前敲定的竞价单分发规则，第一阶段的硬门槛过滤，资质要求为第一优先级，先过滤掉无对应资质的商户，再进行后续的梯队划分；
同评分、同匹配度的情况下，一类资质商户优先进入竞价第一梯队，二类资质次之，三类专项资质仅能参与对应 L1-L2 项目的竞价。
3. 展示环节：资质全量公示，用户一目了然
商户店铺主页、列表页、报价页，必须醒目展示资质等级、可承接的项目复杂度上限、对应专项资质；
高复杂度订单的报价列表，默认优先展示一类资质商户的报价，给用户清晰的决策参考。
4. 评价环节：资质与评价权重强绑定
无对应资质的超范围订单，哪怕私下完成交易，也无法发布评价，不计入店铺评分，彻底杜绝商户超范围接单刷分；
一类资质商户的 L4 订单评价，拥有最高权重，直接决定店铺的顶级排序优先级，形成「合规资质→优质服务→高权重评价→更多精准订单」的正向循环。
四、新能源汽车维修的专项资质补充规则
针对新能源汽车维修，在上述基础上，额外增加专项资质硬门槛，完全贴合《新能源汽车维修服务规范》国家标准：
涉及新能源汽车三电系统（电池、电机、电控）的维修项目，最低按 L3 定级，三电总成拆解大修直接按 L4 定级；
承接新能源汽车三电系统维修项目的商户，除了满足对应等级的资质要求，还必须上传《新能源汽车维修专项资质证书》、持证技师的《新能源汽车维修工职业技能等级证书》，无专项资质的商户，系统自动屏蔽所有新能源三电相关订单；
仅一类维修资质 + 新能源专项资质的商户，可承接新能源汽车三电总成大修、事故车整车修复的 L4 项目。
五、不可突破的合规红线（绝对不能放宽）
很多人会问：行业里很多二类修理厂偷偷接发动机大修、事故车修复，平台能不能放宽？答案是绝对不能，核心原因有 3 个：
法律责任不可规避：平台作为撮合服务提供方，有法定的审核义务，明知商户超范围经营仍提供撮合服务，属于共同违法，一旦出现维修事故、监管核查，平台将面临行政处罚、连带巨额赔偿，甚至吊销经营资质；
用户安全与平台公信力不可牺牲：总成大修、结构件修复，直接关系到用户的行车安全，无一类资质的商户，不具备对应的维修设备、检验能力、技师资质，维修质量无法保障，一旦出事故，平台的公信力会直接崩塌；
原有体系的底层逻辑不可破坏：你整个项目的核心是「真实评价、中立信任」，如果允许无资质商户接高难度订单，就会出现「商户靠超范围接单刷高权重评价，挤压合规优质商户的生存空间」，最终劣币驱逐良币，整个体系彻底崩塌。
