汽修项目复杂度四级分级标准（最终落地版）
本标准 100% 兼容你平台的评价权重、奖励金核算、商户资质门槛、竞价分发权限、评价凭证要求全体系规则，无交叉、无模糊地带，商户和车主可一眼看懂，系统可直接按标准自动匹配对应规则。
一、分级核心原则（不可突破）
技术难度为第一核心：维修项目对技师资质、专业设备、诊断能力的要求越高，复杂度等级越高；
风险权重强绑定：车主的资金投入风险、车辆安全风险越高，复杂度等级越高，对应评价权重、奖励金上限同步提升；
合规资质强匹配：每个等级对应交通部《机动车维修管理规定》的法定维修资质，无对应资质的商户，不可承接对应等级项目的竞价订单；
就高不就低兜底：同一订单包含多个不同等级的维修项目，直接按最高等级匹配规则，杜绝拆分订单降权套利。
二、四级复杂度完整标准
L1 极低复杂度（基础便民服务）
核心定义
无技术门槛、无配件更换 / 仅更换超低价易损件、无车辆安全风险，纯便民类服务，三类维修资质即可承接。
覆盖项目清单
基础洗车、内饰清洁、车辆打蜡 / 封釉等美容项目；
轮胎补气、四轮动平衡（不含四轮定位）；
空调滤芯、空气滤芯更换（无需拆卸核心部件，徒手 / 简单工具即可完成）；
雨刮片、灯泡、保险丝等超低价易损件更换；
全车基础安全检测、电脑故障码读取（不含故障排查与维修）。
核心特征
客单价普遍＜500 元，施工时长＜1 小时，无任何维修风险，普通洗车工即可完成。
平台规则适配
评价权重：0.2
基础奖励金：10 元，单笔封顶 50 元
商户资质门槛：三类及以上维修资质
评价凭证要求：仅需 1 张施工结果实拍图，无其他强制要求
竞价权限：全量合规商户可承接
L2 低复杂度（常规维保 / 易损件更换）
核心定义
低技术门槛、常规标准化更换项目、车辆安全风险极低，需要基础维修技能与专用工具，二类维修资质 / 三类专项维修资质可承接。
覆盖项目清单
常规小保养（机油、机油滤芯更换，含常规全车检测）；
刹车片、刹车盘、刹车油更换；
火花塞、燃油滤芯、变速箱油（重力更换）更换；
电瓶、轮胎更换、四轮定位；
小面积钣金喷漆（单块翼子板 / 车门，不含结构件修复）；
常规易损件更换（下摆臂胶套、稳定杆连杆、减震器顶胶等）；
新能源车辆：常规基础保养、空调系统维修、轮胎 / 电瓶更换、非三电系统常规易损件更换。
核心特征
客单价普遍 500-3000 元，施工时长 1-4 小时，标准化流程，无核心部件拆解，持证初级技师即可完成。
平台规则适配
评价权重：1.0
基础奖励金：30 元，单笔封顶 200 元
商户资质门槛：二类 / 三类专项维修资质及以上
评价凭证要求：仅需 1-2 张新旧配件对比图 / 施工结果图
竞价权限：全量合规商户可承接
L3 中复杂度（常规故障维修 / 专项整备）
核心定义
中等技术门槛、需要专业故障诊断能力、核心部件局部拆解维修、存在一定的车辆安全风险，必须二类及以上维修资质可承接。
覆盖项目清单
发动机局部维修（气门室盖垫更换、正时套装更换、水泵 / 节温器更换、喷油嘴 / 节气门清洗维修等，不含发动机拆解大修）；
变速箱局部维修（阀体更换、变速箱油循环机更换、油封更换等，不含变速箱拆解大修）；
底盘系统整备（减震器、下摆臂、转向机、元宝梁等核心底盘件更换维修）；
制动系统深度维修（ABS 泵、刹车总泵 / 分泵维修更换）；
车身电气系统深度维修（车身电脑、空调压缩机、发电机、起动机维修更换）；
中大面积钣金喷漆（3 个及以上漆面，不含结构件修复、整车喷漆）；
新能源车辆：三电系统基础检测与故障排查、非核心三电部件维修、电池包均衡与常规维护、车载充电机 / DC-DC 转换器更换维修。
核心特征
客单价普遍 3000-10000 元，施工时长 4-24 小时，需要专业诊断设备与中级持证技师，涉及车辆核心系统拆解，对维修精度有明确要求。
平台规则适配
评价权重：3.0
基础奖励金：100 元，单笔封顶 800 元
商户资质门槛：二类及以上维修资质，需有对应项目的持证技师
评价凭证要求：必传 2 张图（维修结果图 + 新旧核心配件对比图 / 维修明细单）
竞价权限：仅二类及以上资质商户可承接，一类资质商户优先进入第一梯队
L4 高复杂度（总成大修 / 整车修复 / 疑难故障）
核心定义
高技术门槛、需要高级技师资质与专业成套设备、涉及车辆核心总成拆解大修 / 整车结构修复、极高的资金与安全风险，必须一类维修资质可承接。
覆盖项目清单
发动机总成拆解大修、缸体修复、发动机重组；
变速箱总成拆解大修、变矩器维修、变速箱重组；
事故车整车修复（含车身结构件、纵梁 / ABC 柱修复、整车钣金喷漆）；
车辆泡水车全流程维修、整车电路系统重构；
疑难故障排查（多次维修未解决、无明确故障码的隐性故障、多系统联动故障）；
新能源车辆：三电系统总成大修（电池包拆解维修 / 电芯更换、驱动电机大修、电机控制器维修）、新能源整车事故车修复、三电系统疑难故障排查。
核心特征
客单价普遍 10000 元以上，施工时长 3 天以上，需要高级 / 专项持证技师、成套专业维修设备，对维修资质与技术能力有法定强制要求。
平台规则适配
评价权重：6.0（保险事故车订单权重 ×2）
基础奖励金：300 元，单笔封顶 2000 元（保险事故车专属封顶 3000 元）
商户资质门槛：一类维修资质，需有对应项目的高级持证技师与专项设备
评价凭证要求：必传 2 张核心图（维修结果图 + 新旧核心配件对比图 / 维修明细单 / 定损单），过程图为加分项
竞价权限：仅一类资质商户可承接，主机厂授权 4S 店 / 品牌专修厂优先进入第一梯队
三、分级判定兜底规则
跨等级订单判定：同一订单包含多个不同等级的维修项目，直接按最高等级的项目匹配全订单规则，不得拆分订单、分开发布；
模糊项目判定：未明确列入清单的维修项目，按「技术难度、客单价、安全风险」三个维度，由系统自动匹配对应等级，有争议的订单由人工复核最终定级；
资质一票否决：无对应等级法定维修资质的商户，不可承接对应等级的竞价订单，系统自动过滤，杜绝超资质接单；
新能源项目专项判定：涉及三电系统的维修项目，最低按 L3 定级，三电总成大修直接按 L4 定级，需额外提供新能源专项维修资质，无资质商户自动过滤。
